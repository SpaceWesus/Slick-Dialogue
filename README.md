# Dialogue Log

Small RE_Kenshi plugin that adds a draggable transparent dialogue overlay.

## Files

- `DialogueLog.dll`
- `RE_Kenshi.json`

## Requirements

- Kenshi 1.0.65
- RE_Kenshi

## Notes

This first build captures `Dialogue::say(text, line)` and appends the most recent dialogue lines to a movable window near the middle-right of the screen.
